#  Medisphere – Your Health Companion

Medisphere is a frontend web project I built as a first-year student to practice web development.
It is a health-based platform that helps users manage medicines, check safety, and keep track of their health in one place.



##  About the Project

This is one of my early projects where I tried to build something useful and not just a basic website.

The idea behind Medisphere is to solve small real-life problems like:

* remembering medicines
* finding cheaper alternatives
* checking if food/medicine is safe

The project mainly focuses on frontend functionality and user experience.



##  Features

*  Login & Signup system (demo-based)
*  Dashboard with multiple health tools
*  Camera Scanner (simulated safety check)
*  Doctor Queue System (availability + wait time)
*  Generic Medicine Finder (cost comparison)
*  Pill Reminder (track medicines)
*  Medical Records (upload & view documents)
*  Profile section (health details)



##  Tech Used

* HTML
* CSS
* JavaScript

Everything is built using **pure frontend (no frameworks)** to strengthen my basics.
